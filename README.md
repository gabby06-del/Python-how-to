# Python: From Basics to Building Real Software

Welcome! This repository is a self-paced journey through Python and core computer science concepts - designed to take anyone from complete beginner to writing well-structured, large-scale software with hands on learning

## 🎯 Goal

To teach Python from the ground up, one lecture at a time, while building real intuition for how software is actually designed and built

## 📚 Structure

- **Lectures** cover concepts sequentially, starting with the fundamentals (variables, strings, loops, functions, objects) and gradually progressing to more advanced topics (data structures, algorithms, object-oriented design, and large-scale software architecture).
- **Every 5 lectures**, a hands-on **project** ties together everything learned so far. These projects are meant to demonstrate practical, applicable skills, not just theory.

## 🗂️ Topics Covered (Roadmap)

- Rudimentary CS concepts: variables, strings, data types, control flow
- Functions and scope
- Objects and classes (OOP fundamentals)
- Data structures (lists, dictionaries, sets, tuples)
- File handling and error handling
- Algorithms and problem-solving patterns
- Intermediate design patterns
- Large-scale software design and project structure
- Testing, debugging, and best practices

*(This roadmap will grow and evolve as the project progresses.)*

This isn't just a collection of scripts — it's a structured learning path meant to build both **Python fluency** and **software design thinking**, so that by the end, the skills learned are genuinely applicable to real-world programming

## 📖 Course Pacing & Inspiration

The pacing and structure of this project draw heavily from two well-established computer science curricula:

- **[OSSU Computer Science](https://github.com/ossu/computer-science)** — an open-source, community-driven curriculum that mirrors a university-level CS education.
- **[CS50 (Harvard's Introduction to Computer Science)](https://cs50.harvard.edu/)** — for its approach to teaching foundational CS concepts in a structured, project-driven way.

This repo isn't a direct copy of either — the general topic sequencing and pacing take inspiration from how these courses structure a CS education. However, **all lecture notes, exercises, projects, and code examples in this repository are my own original work**, written to reflect my own understanding and learning process as I go.
